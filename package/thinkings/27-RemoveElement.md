# 移除元素

给你一个数组 nums 和一个值 val，你需要 **原地** 移除所有数值等于 val 的元素，并返回移除后数组的新长度。

不要使用额外的数组空间，你必须仅使用 O(1) 额外空间并 **原地** 修改输入数组。

元素的顺序可以改变。你不需要考虑数组中超出新长度后面的元素。

说明:

为什么返回数值是整数，但输出的答案是数组呢?

请注意，输入数组是以「引用」方式传递的，这意味着在函数里修改输入数组对于调用者是可见的。

你可以想象内部操作如下:

```c
// nums 是以“引用”方式传递的。也就是说，不对实参作任何拷贝
int len = removeElement(nums, val);

// 在函数里修改输入数组对于调用者是可见的。
// 根据你的函数返回的长度, 它会打印出数组中 该长度范围内 的所有元素。
for (int i = 0; i < len; i++) {
    print(nums[i]);
}
```

示例 1：

```js
输入：nums = [3,2,2,3], val = 3
输出：2, nums = [2,2]
解释：函数应该返回新的长度 2, 并且 nums 中的前两个元素均为 2。你不需要考虑数组中超出新长度后面的元素。例如，函数返回的新长度为 2 ，而 nums = [2,2,3,3] 或 nums = [2,2,0,0]，也会被视作正确答案。
```

示例 2：

```js
输入：nums = [0,1,2,2,3,0,4,2], val = 2
输出：5, nums = [0,1,4,0,3]
解释：函数应该返回新的长度 5, 并且 nums 中的前五个元素为 0, 1, 3, 0, 4。注意这五个元素可为任意顺序。你不需要考虑数组中超出新长度后面的元素。
```

提示：

- 0 <= nums.length <= 100
- 0 <= nums[i] <= 50
- 0 <= val <= 100

思路：

拿到这个题目，因为要求原地修改数组并且不能使用额外空间，所以很自然想到使用双指针的方法。通过一个指针遍历数组，另一个指针标记新数组的位置，将不等于目标值的元素依次放到新位置，从而实现原地删除元素的效果。选择这种方法是因为它能在不创建新数组的情况下，高效地完成元素的筛选和重新排列。

1. 首先判断数组是否为空，如果为空则直接返回 0。这是一个必要的边界情况处理。
2. 初始化 `res` 为 0，用于标记新数组的起始位置。
3. 使用循环遍历数组，对于每个元素，如果其不等于目标值 `val`，就将其赋值到新位置 `nums[res]`，然后将 `res` 递增。
4. 最终返回 `res`，即为新数组的长度。

时间复杂度：O(n)，其中 n 是数组 `nums` 的长度。因为需要遍历数组一次。
空间复杂度：O(1)，只使用了固定的额外变量 `res` 和 `i`，没有额外的空间与输入规模成正比。

```js
/**
 * @param {number[]} nums
 * @param {number} val
 * @return {number}
 */
const removeElement = (nums, val) => {
  if (nums.length === 0) return 0;
  let res = 0;

  for (i = 0; i < nums.length; i++) {
    if (nums[i] !== val) {
      nums[res] = nums[i];
      res += 1;
    }
  }
  return res;
};
```

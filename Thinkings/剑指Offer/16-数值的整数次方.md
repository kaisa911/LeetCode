# 数值的整数次方

实现 pow(x, n) ，即计算 x 的 n 次幂函数（即，xn）。不得使用库函数，同时不需要考虑大数问题。

示例 1：

```js
输入：x = 2.00000, n = 10
输出：1024.00000
```

示例 2：

```js
输入：x = 2.10000, n = 3
输出：9.26100
```

示例 3：

```js
输入：x = 2.00000, n = -2
输出：0.25000
解释：2^-2 = 1/2^2 = 1/4 = 0.25
```

提示：

- -100.0 < x < 100.0
- -2^31 <= n <= 2^31-1
- -10^4 <= xn <= 10^4

**思路：**

```ts
function myPow(x: number, n: number): number {
  let res: number = 1.0;
  let t: number = n;
  while (n) {
    if (n & 1) res *= x;
    x *= x;
    n /= 2;
  }
  return t > 0 ? res : 1.0 / res;
}

```
